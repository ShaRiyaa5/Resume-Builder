# Resume Builder

Streamlit app where users fill in their details and get a formatted resume - covers personal info, education, experience, and skills with a clean output.

## Tech Stack
Python · Streamlit · FPDF

## Run Locally
pip install -r requirements.txt
streamlit run resume_builder.py

## What it does
- Dynamic form inputs for all resume sections
- Generates a cleanly formatted downloadable resume
- Modular code structure - easy to extend
